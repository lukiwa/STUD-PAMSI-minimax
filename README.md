# STUD_PAMSI_minimax
Connect4 game with simple AI - MiniMax with Alpha-Beta pruning.

## To compile:
1. Clone this repository
2. cd STUD_PAMSI_minimax
3. mkdir release
4. cd release
5. cmake ..
6. make
7. Launch program - ./STUD_PAMSI_minimax
